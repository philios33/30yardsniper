== Description ==
<br />
[Author Site](http://onlinewebapplication.com)|
[Plugin Home Page](http://onlinewebapplication.com/2011/10/wp-ourstats-widget.html)
<br />
<br />
ourSTATS.de is a 100% free Service to find out information about your visitors such as eg Screen resolution, browser, language and origin, page views by hour survey, as well as what keywords your visitors use to go to your homepage.

This plugin creates a widget that displays the ourstats.de counter code on your blog. You can always change the button style counter.
Look at: http://www.ourstats.de/

== Installation ==

This section describes how to install the plugin and get it working.

Manual installation:

1. Upload the `ourstatsde-wp-widget` folder to the `/wp-content/plugins/` directory

Installation using "Add New Plugin"

1. From your Admin UI (Dashboard), use the menu to select Plugins -> Add New
2. Search for 'ourSTATS WP Widget'
3. Click the 'Install' button to open the plugin's repository listing
4. Click the 'Install' button

Activation and Use

1. Activate the plugin through the 'Plugins' menu in WordPress
2. Drag&Drop the WP ourSTATS Widget in your sidebar 
3. Configure settings, and save
